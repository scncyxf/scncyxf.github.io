一、什么是Hexo

Hexo是一个快速、间接且高效的博客框架。Hexo使用Markdown(或其他标记语言来解析文章)，在几秒内即可利用靓丽的主题生成静态网页

二、安装前提

安装Hexo需要先安装下列应用程序

  • Node.js(建议使用Node.js 12.0及以上版本)

  • Git

1. git

  • 安装git

2. node.js

  • 安装node.js
  • 换源

三、安装Hexo

所有必备的程序安装好后，即可使用npm安装Hexo

四、建站

  • 安装Hexo完成后，执行下列命令，Hexo会在指定文件夹新建所需要的文件按

  • 文件用途

    • _config.yml，网站的配置信息

    • package.json，应用程序的信息

    • scaffolds， 模板文件夹

    • source，存放用户资源的地方

    • themes， Hexo会根据主题生成静态页面

五、配置

  • 安装主题

  • 配置主题

    参考每个主题的wiki

六、部署

  • 注册github账号，建立仓库，名称为：scncyxf.github.io

  • 安装hexo-deployer-git插件

  • 修改config.yml文件(注：https访问目前只支持token，需要提前设置)

  • 执行hexo命令

  • 访问地址： scncyxf.github.io
