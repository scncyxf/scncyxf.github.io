git使用
2023-08-08|常用软件

git基本概念：

  1. Workspace：工作区，平时存放项目代码的位置
  2. Index/stage：暂存区，临时存放文件改动
  3. Repository：仓库区，存放发布的各个版本
  4. Remote：远程仓库

  2. 安装完成并打开

  4. 查看配置

三、添加ssh密钥

  1. 使用git软件生成密钥文件id_rsa和id_rsa.pub

  2. 复制公钥id_rsa.pub，添加到github网站setting中ssh and GPG keys中

  3. 验证是否连接成功

    显示内容如下，则表示连接成功：

    Hi scncyxf! You’ve successfully authenticated, but GitHub does not provide shell access
